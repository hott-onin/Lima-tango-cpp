Device Commands Description

LimaDetector Class

Revision: - Author: arafat.noureddine



1 - Init

  • Description: This commands re-initialise a device keeping the same network connection.
    After an Init command executed on a device, it is not necessary for client to re-connect to the device.
    This command first calls the device delete_device() method and then execute its init_device() method.
    For C++ device server, all the memory allocated in the nit_device() method must be freed in the delete_device() method.
    The language device desctructor automatically calls the delete_device() method.
     
  • Argin:
    DEV_VOID     : none.
     
  • Argout:
    DEV_VOID     : none.
     
  • Command allowed for:
    • Tango::INIT
    • Tango::STANDBY
    • Tango::FAULT
    • Tango::RUNNING

     


2 - State

  • Description: This command gets the device state (stored in its device_state data member) and returns it to the caller.
     
  • Argin:
    DEV_VOID     : none.
     
  • Argout:
    DEV_STATE     : State Code
     
  • Command allowed for:
    • Tango::INIT
    • Tango::STANDBY
    • Tango::FAULT
    • Tango::RUNNING

     


3 - Status

  • Description: This command gets the device status (stored in its device_status data member) and returns it to the caller.
     
  • Argin:
    DEV_VOID     : none.
     
  • Argout:
    CONST_DEV_STRING     : Status description
     
  • Command allowed for:
    • Tango::INIT
    • Tango::STANDBY
    • Tango::FAULT
    • Tango::RUNNING

     


4 - Prepare

  • Description: Prepare the acquisition (Apply parameters like bin/roi/exposure/.. & allocate buffers & ...)
     
  • Argin:
    DEV_VOID     :
     
  • Argout:
    DEV_VOID     :
     
  • Command allowed for:
    • Tango::STANDBY
    • Tango::RUNNING

     


5 - Snap

  • Description: Starts the acquisition of a number of frames equal to 'nbFrames' attribute value.
     
  • Argin:
    DEV_VOID     :
     
  • Argout:
    DEV_VOID     :
     
  • Command allowed for:
    • Tango::STANDBY
    • Tango::RUNNING

     


6 - Start

  • Description: Starts a "video/live" acquisition of an infinite number of frames.
    In this mode, In this mode, the frame loss is permitted and not signaled as FAULT in the device state
    It is not allowed to generate files in this mode.
     
  • Argin:
    DEV_VOID     :
     
  • Argout:
    DEV_VOID     :
     
  • Command allowed for:
    • Tango::STANDBY
    • Tango::RUNNING

     


7 - Stop

  • Description: Stop current acquisition/video.
     
  • Argin:
    DEV_VOID     :
     
  • Argout:
    DEV_VOID     :
     
  • Command allowed for:
    • Tango::STANDBY
    • Tango::RUNNING

     


8 - SetROI

  • Description: Define a Region of Interest . (OriginX, OriginY, Width, Height)
    Roi and Binning are associated.
     
  • Argin:
    DEVVAR_ULONGARRAY     : [origin_x, origin_y, width, height]
     
  • Argout:
    DEV_VOID     :
     
  • Command allowed for:
    • Tango::STANDBY

     


9 - SetBinning

  • Description: Define a binning Horizontal (x) & Vertical (y) for the image.

     
  • Argin:
    DEVVAR_ULONGARRAY     :
     
  • Argout:
    DEV_VOID     :
     
  • Command allowed for:
    • Tango::STANDBY

     


10 - ResetBinning

  • Description: Use the binning Hrizontal=1 & Verical=1 of the detector according to current Roi.
     
  • Argin:
    DEV_VOID     :
     
  • Argout:
    DEV_VOID     :
     
  • Command allowed for:
    • Tango::STANDBY

     


11 - ResetROI

  • Description: Use the full frame of the detector according to current Binning.
     
  • Argin:
    DEV_VOID     :
     
  • Argout:
    DEV_VOID     :
     
  • Command allowed for:
    • Tango::STANDBY

     


12 - GetAttributeAvailableValues (for expert only)

  • Description: Return a list of string containing all available values of a DevString attribute.
     
  • Argin:
    DEV_STRING     : Attribute name
     
  • Argout:
    DEVVAR_STRINGARRAY     : List of strings containing the available values
     
  • Command allowed for:
    • Tango::STANDBY

     


13 - GetAvailableCapabilities

  • Description: Return a list of string containing all available capabilitiesof the detector :
    - Roi
    - Binning
    - Trigger
    - Shutter
    - Event Errors
    - ...

     
  • Argin:
    DEV_VOID     :
     
  • Argout:
    DEVVAR_STRINGARRAY     : The list of available capabilities
     
  • Command allowed for:
    • Tango::STANDBY

     


14 - ResetFileIndex (for expert only)

  • Description: Reset the file index
     
  • Argin:
    DEV_VOID     :
     
  • Argout:
    DEV_VOID     :
     
  • Command allowed for:
    • Tango::STANDBY

     


15 - ReloadROI

  • Description: This command allows reloading the last ROI values set using the SetROI command.
     
  • Argin:
    DEV_VOID     :
     
  • Argout:
    DEV_VOID     :
     
  • Command allowed for:
    • Tango::STANDBY

     


16 - GetDataStreams

  • Argin:
    DEV_VOID     :
     
  • Argout:
    DEVVAR_STRINGARRAY     :
     
  • Command allowed for:
